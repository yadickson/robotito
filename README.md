# robotito

Un jueguito para aprender a mover un robot en un tablero

## required

- autoconf
- automake
- libtool
- pkg-config
- libcpputest-dev
- libncurses-dev
- lcov
- cppcheck
- clang-format

## configuration

```bash
autoreconf -ivf
```

```bash
./configure
```

## build

```bash
make
```

## run

```bash
./src/robotito
```
